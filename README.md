# BarrioBook-Automation
Proyecto base: BarrioBook  
Tecnologías: ASP.NET Core, SQLite, Selenium WebDriver, NUnit y ExtentReports

## Descripción
BarrioBook es un sistema de gestión de micro-librería. Este repositorio incluye:
- la aplicación BarrioBook
- pruebas automatizadas con Selenium
- reporte HTML de pruebas
- capturas automáticas

## Funcionalidades automatizadas
- Login de administrador
- Crear libro
- Consultar libros
- Actualizar libro
- Eliminar libro

## Credenciales de prueba
- Email: admin@barriobook.com
- Password: Admin123*

## Cómo ejecutar
1. Ejecutar BarrioBook.API
2. Abrir BarrioBook.Client con Live Server
3. Ejecutar las pruebas desde BarrioBook.SeleniumTests

## Evidencias
- Reportes HTML en la carpeta Reports
- Capturas automáticas en la carpeta Screenshots
